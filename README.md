# Small Trade Manager
Projeto destinado à disciplina "Construção de Software".

### Frameworks Utilizados
- [Bootstrap](https://getbootstrap.com/)
- [Chart.js](https://www.chartjs.org/)
- [jQuery](https://jquery.com/)
- [jQuery Mask Plugin](https://igorescobar.github.io/jQuery-Mask-Plugin/)
- [printThis](https://jasonday.github.io/printThis/)

### Instruções
Crie um banco de dados no SGBD SQL utilizado e insira os códigos dos arquivos do diretório `db` seguindo a ordem `tables.sql`, `indexes.sql`, `views.sql`. Depois configure as constantes `MYSQL_HOST`, `MYSQL_SCHEMA`, `MYSQL_USER` e `MYSQL_PASSWORD` do arquivo `config.php` com os dados de acesso ao SGBD.
